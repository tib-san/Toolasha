# Toolasha Feature List

Complete list of all Toolasha features with brief descriptions.

## Action Panel Enhancements

- **Total Action Time** - Shows queue completion time top-left
- **Action Panel Total Time** - Displays time/exp/level goals on panels
- **Quick Input Buttons** - Hour/count preset buttons (1/10/100/Max)
- **Foraging Profit** - Shows profit for multi-outcome actions
- **Action Queue Time** - Total time in queue menu
- **Output Totals** - Calculates expected outputs from quantity
- **Max Produceable** - Shows craftable count on actions

## Item Tooltips

- **Market Prices** - 24-hour average ask/bid prices
- **Production Profit** - Cost and profit calculations
- **Detailed Profit Breakdown** - Materials table with prices
- **Expected Value** - EV for openable containers
- **Consumable Stats** - HP/MP restore speed and efficiency
- **Enhancement Simulator** - Cost calculations for enhancement paths

## Enhancement System

- **Enhancement Tracker** - Track attempts, costs, and statistics
- **Enhancement Simulator** - Calculate optimal enhancement strategies
- **Enhancement Path Display** - Show costs for upgrades

## Economy & Inventory

- **Net Worth Display** - Shows total assets top-right
- **Inventory Summary** - Worth breakdown below inventory
- **Inventory Sort** - Sort by value/type
- **Stack Value Badges** - Show value on item icons
- **Pricing Modes** - Conservative/Hybrid/Optimistic profit calculations

## Combat Features

- **Combat Score** - Gear score on profile panel
- **Zone Indices** - Index numbers on zones
- **Ability Book Calculator** - Books needed for levels

## Tasks

- **Task Profit Display** - Total profit for tasks
- **Task Reroll Tracker** - Tracks reroll costs
- **Task Zone Index** - Shows zone numbers

## UI Enhancements

- **Skill XP Percentage** - Shows % to next level
- **Equipment Level Display** - Shows level on icons
- **Key Zone Index** - Shows zone on key icons
- **Alchemy Item Dimming** - Dims unavailable items
- **Market Filter** - Filter by level/class/slot
- **Auto-Fill Market Price** - Fills optimal market prices

## House

- **House Upgrade Costs** - Shows costs with inventory comparison

## Settings & Configuration

- **Network Alert** - Alert when data fetch fails
- **Enhancement Simulator Config** - Skill/tea/tool settings
- **Pricing Mode Selection** - Choose profit calculation method
- **Expected Value Options** - Drop display preferences
- **Inventory Badge Options** - Badge display settings
- **Color Customization** - Lots of color customizability
